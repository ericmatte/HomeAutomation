## Atrium Dashboard

The Atrium dashboard is a fully dynamic Lovelace dashboard generated by a custom strategy: floors, rooms, and entities are read from Home Assistant at load time, so adding an area or a light in HA is enough to update the UI — no YAML edits required.

JS sources live in [`www/atrium/`](www/atrium/):

- `strategy.js` — dashboard strategy + header + floor label (single entry point; imports the others as ES modules)
- `area-card.js` — per-floor area renderer
- `popover.js` — shared popover component

### Prerequisites

1. Install [HACS](https://hacs.xyz/) if you don't have it yet.
2. Through HACS → Frontend, install [mini-graph-card](https://github.com/kalkih/mini-graph-card) (used to render the climate background graph on each room tile).
3. Copy the `www/atrium/` folder from this repo into your Home Assistant `/config/www/atrium/` directory (file editor add-on, Samba, or `scp`).

### Setup in the UI

1. Go to **Settings → Dashboards → Resources** (top-right ⋮ menu → Resources if hidden).
2. Add a new resource:
   - URL: `/local/atrium/strategy.js`
   - Resource type: **JavaScript module**
3. Reload the browser (hard refresh) so HA picks up the new resource.
4. Go to **Settings → Dashboards → Add Dashboard → New dashboard from scratch**.
   - Title: `Atrium` (or anything you like)
   - Icon: `mdi:home-variant`
   - Show in sidebar: on
5. Open the new dashboard, then **⋮ (top-right) → Edit dashboard → ⋮ → Raw configuration editor**, replace the content with:

   ```yaml
   strategy:
     type: custom:atrium
   views: []
   ```

6. Save. The dashboard rebuilds itself on every load from your HA floors/areas/entities.

### Xiaomi Roborock

In order to be able to connect the Roborock to Home Assistant, you first need to get the token.

This can be done by downloading the version 5.4.49 of the Mi Home App, and then checking the logs:

- https://www.home-assistant.io/components/vacuum.xiaomi_miio/#retrieving-the-access-token
- https://www.apkmirror.com/apk/xiaomi-inc/mihome/mihome-5-5-49-release/mi-home-5-5-49-android-apk-download/download/
